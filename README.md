Ticket Booking Platform - README & Problem Statement
 1. Overview
 This project is a simple Python-based ticket booking platform.
 Users can select a movie, choose seats, check availability, and confirm booking.
 2. Features- Movie list display- Seat availability check- Seat booking system- Prevent double-booking- User-friendly menu system
 3. How to Run
 1. Install Python.
 2. Save the code as ticket_booking.py
 3. Run using: python ticket_booking.py
 4. Problem Statement
 Create a Python program that:- Shows available movies- Shows available seats- Allows the user to book seats- Stores booked seats so they cannot be booked again- Displays confirmation after booking
 5. Example Flow
 Movies: Pathaan, Jawan, Leo
 User selects: Jawan
Seats available: A1, A2, A3
 User books: A2
 Output: Seat A2 successfully booked!
